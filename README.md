<h2>依赖</h2>
<ul>
<li>Python</li> 
<li>主流Linux发行版</li>
</ul>
<h2>安装</h2>
首先，从github上下载，可以直接利用git clone，也可以下载压缩包自己解压然后安装。然后，从项目中clone下来并安装</br>
cd YaH3C</br>sudo python setup.py install<br>
